Описание программы

Программа служит для проверки орфографии (написания слов) английского языка.
Во время проверки пользователь вводит слова, используя перевод, описание (транскрипцию) или на слух (если слово озвучено). Программа проверяет введенные слова по словарю на ошибки.
При наличии ошибок эти слова нужно будет снова ввести в следующем проходе проверки.
Особенностью метода является то, что при вводе слова, в правильном написании которого нет уверенности, можно установить флаг «Не уверен». Такие слова, вместе со словами с ошибками, переходят в следующий проход проверки. Пометку «Не уверен» у слова следует ставить в следующих проходах до тех пор, пока написание слова не запомнится. Проверка завершается тогда, когда в новом проходе не осталось слов с ошибками или с пометкой «Не уверен».

Ниже приводится краткое описание работы с программой.
Более подробная информация содержится в начале каждой страницы программы в разделе "Описание страницы" (раскрывается нажатием на кнопку "+").

Регистрация

Для работы в программе следует зарегистрироваться (в меню по ссылке «Зарегистрироваться»). В форме регистрации вводится логин (должен быть уникальным), имя, пароль и (необязательно) e-mail.
В дальнейшем можно изменить реквизиты пользователя в меню "Изменить реквизиты", а сменить пароль - в меню "Сменить пароль".

Вход в программу

Если пользователь уже зарегистрирован, то перед началом работы следует осуществить вход в программу (в меню по ссылке «Войти») введя логин и пароль.

Проверки

Для проверки орфографии нужно зайти на страницу проверок (в меню ссылка «Проверки») и создать новую проверку (или продолжить старую).
При создании новой проверки нужно будет выбрать один из доступных словарей. После этого кнопкой «Начать проверку» можно начать первый проход проверки орфографии.
В ходе проверки предлагается прослушать слово и прочитать (если есть) перевод, описание (транскрипцию) слова и ввести слово.
Если нет уверенности в правильном написании слова, следует поставить флаг «Не уверен». В этом случае это слово будет предложено ввести снова в следующем проходе.
Введенное слово будет проверено на ошибки и будет показано правильное написание слова из словаря. Слова с ошибками также перейдут в следующий проход.
Если ошибка допущена в самом словаре, то можно отметить это кнопкой «Указать на ошибку в словаре». Отметить в чем ошибка (в произнесении, переводе, описании) или написать комментарий. Автор словаря сможет просмотреть указанные ошибки и исправить их.
Слова с ошибками в словаре (или слова, которые незнакомы) можно пропустить (кнопкой «Пропустить слово»). Данные слова не перейдут в следующий проход. После завершения прохода проверки будет предложено перейти к следующему проходу. Если в проходе слов с ошибками или слов с пометкой «Не уверен» не осталось, то следует завершить проверку.
При осуществлении первого прохода проверки можно указать порядок слов (случайный, по алфавиту), исключить слова со звуком или слова без звука, исключить слова уже пройденные в предыдущих завершенных проверках.
В первом проходе также доступен режим «Тренировка», в котором показывается написание слова и установлена пометка «Не уверен». Режим нужен в случае, если в словаре много незнакомых слов (слов которые ни разу не использовались в письме).

Словари

В программе есть возможность создавать и озвучивать свои собственные словари. Для этого следует перейти на страницу словарей (в меню ссылка «Словари») и добавить новый словарь.
Кнопкой «Редактировать» в списке словарей можно перейти на страницу редактирования словаря.
На странице словаря можно добавлять слова по одному в словарь и записывать звук.
Для записи звука требуется наличие подключенного микрофона. В браузере должен быть установлен «Adobe Flash Player». В окне «Adobe Flash» следует разрешить использование микрофона.
Более удобный способ создания словаря заключается в загрузке слов из текста или из списка слов. После загрузки слов в словарь можно озвучить слова (кнопка «Озвучить»).
Для того, что бы вашим словарем могли воспользоваться другие пользователи нужно у словаря установить флаг «Общий». Можно разрешить использовать словарь (или редактировать) определенным пользователям добавив им права на словарь.
